# VKey

VKey is a lightweight virtual keyboard for macOS.

The app lives in the menu bar, opens a custom `NSPanel`, and provides a compact on-screen keyboard with support for alternative character popups.

## Features

- Menu bar app with a status bar icon
- Custom floating `NSPanel`
- Alternative character popup rendered in a top-level overlay
- Language switching
